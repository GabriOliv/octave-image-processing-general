### Usando os Scripts:

* Unzip os scripts
* Selecione a pasta no ambiente de trabalho do [Octave](https://www.gnu.org/software/octave/index)
* Chame no Terminal como no Exemplo

Exemplo 01:
```matlab
>> equali_img_save(imread('001_img.jpg'));
```
Exemplo 02:
```matlab
>> A = imread('001_img.jpg');
>> equali_img_save(A);
```
